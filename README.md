--- PROJECT OVERVIEW ---
This is a futuristic, House of the Dragon themed web 
interface. It merges ancient Valyrian lore with a 
high-tech "Cyberpunk" aesthetic.

--- CORE TERMINALS (PAGES) ---

1. FRONT TERMINAL (index.html)
   - Entry point to the dynasty archives.

2. THE DRAGON VAULT (dragons.html)
   - Features the DNA Scanner. 
   - Uses birth-month algorithms to match riders to dragons.
   - Displays real-time threat levels and combat history.

3. THE ROYAL DYNASTY (house.html)
   - Loyalty Filter System: Toggle between TEAM BLACK and TEAM GREEN.
   - Intel Panels: Access hidden stats on weapons and DNA purity.

4. THE CHRONICLES (history.html)
   - Historical data storage for the Seven Kingdoms.

--- TECHNICAL SPECS ---
- Languages: HTML5, CSS3, JavaScript (Vanilla)
- Icons: FontAwesome v6.5.1
- Styling: Custom Glow Effects, DNA Scan-line animations
